# Recoeve.net Personal Info Handle We only collect your e-mail. Password is strongly encrypted by hashing 10,000 times with a random salt per one user. In network flow, we also encrypt the password with decreasing hash. So no one can break our security system. 우리는 당신의 이메일만을 수집합니다. 암호/비밀번호는 강력하게 보안하여 저장하고 있습니다. 10,000번의 hash 를 돌리고 유저별로 랜덤한 salt 를 추가하여 강력하게 그 누구도, 심지어 양자 컴퓨터도 못뚫도록 보안을 철저히 하였습니다. 네트워크 단에서도 암호를 감소하는 hash 를 통해 철저히 그 누구도 우리의 안전 시스템을 못뚫도록 보안에 신경쓰고 있습니다. ## PH
